# Phishing Campaign Simulation for Awareness Training

## Disclaimer
This laboratory project is developed strictly for educational purposes and security awareness training simulation. No credential harvesting, data logging, or malicious exploitation methods are implemented. Executing simulation concepts on systems without explicit authorization is strictly prohibited.

* **Phishing Staging Platform:** OS: Kali GNU/Linux Rolling (Version 2026.2)
* **Target / Client Machine:** OS: Ubuntu Desktop
