# File Encryption and Decryption Laboratory Practice

## Disclaimer
This project is developed for educational and security research purposes only. The objective is to understand cryptographic concepts, symetric key management, and data preservation mechanics. Unauthorized encryption of data on systems without prior written consent is strictly illegal.

## Lab Environment Specifications
* **Hypervisor:** VMware Fusion 26H1
* **Development / Attacker Machine:** OS: Kali GNU/Linux Rolling (Version 2026.2)
* **Target / Target Machine:** OS: Ubuntu Desktop
