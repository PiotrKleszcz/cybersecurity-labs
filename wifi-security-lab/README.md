# Wireless Security Assessment and Hardening Laboratory (iPhone Hotspot Sandbox)

## Disclaimer
This laboratory project is conducted strictly for educational purposes and localized wireless security auditing. All testing procedures are performed within a controlled sandbox environment utilizing a personal iOS mobile hotspot as the target Access Point (AP). The operator maintains 100% ownership and explicit authorization over all assets involved.

## Lab Environment Specifications
* **Hypervisor:** VMware Fusion 26H1 (Bridged Network Mode)
* **Auditing Platform:** OS: Kali GNU/Linux Rolling (Version 2026.2)
* **Target Workstation (Victim):** OS: Ubuntu Desktop
* **Target Access Point (AP):** iOS Personal Hotspot (WPA2-PSK, Vulnerable Passphrase Configuration)
